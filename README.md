# RNA-seq Analysis of ASD vs Unaffected Siblings (GSE212645)

## Overview
Differential expression analysis of human blood RNA-seq data comparing 
individuals with Autism Spectrum Disorder (ASD) to unaffected siblings.

## Dataset
- **Source:** GEO Accession GSE212645
- **Samples:** 40 human blood samples
- **Comparison:** ASD vs unaffected siblings

## Tools & Languages
- R (DESeq2, ggplot2)
- Python (pandas, matplotlib)

## Project Structure
- `data/` — raw and processed count matrices
- `scripts/` — R and Python analysis scripts
- `results/` — figures and tables
- `notebooks/` — exploratory analysis
